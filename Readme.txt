U folderima About , Contact , Home , Work se nalaze svi potrebni fajlovi za pokretanje projekta. Html stranice u folderima About i Home  se mogu pokretati i sa lokalnog kompjutera dok je za rad stranica Contact i Work potrebno projekat staviti na Web Server sa instaliranim php-om.
Napomena : Kako biramo opcije iz menija (all print photography web applications ) na Work stranici aplikacija ucitava slike iz istoimenih foldera i prikazuje ih.

Projekat je radjen sa sljedecom konfiguracijom:
Operativni sistem : Linux
Web Server : Apache sa instaliranim Php-om.
Browser : Firefox
